## What is this?

A realtime demo using WebSockets where the server pushes messages every second
and echoes client messages.

## Why WebSockets?

- Realtime dashboards
- Chat apps
- Live stock prices
- Multiplayer games

## Key concepts

- Persistent connection
- Bidirectional messaging
- Connection lifecycle
- Manual reconnect responsibility

## When NOT to use WebSockets

- Simple request/response
- Low-frequency updates
- Cacheable data
